# TypeScript 基础入门

# 搭建环境

# 全局安装

npm install - g typescript

检测安装是否成功(如果未出现错误说明安装成功)
tsc

# 开始操作

创建dome.ts文件

function dome(){
    let web:string = 'hello world'
    console.log(web)
}
dome()

代码运行

提示

因为是单独代码文件，所以需要在终端使用node执行代码文件，如果没有node环境的，请先进行安装node环境

运行命令

node dome.ts
执行完之后会报错， 使用命令把 `dome.ts`
进行转换

tsc dome.ts
执行完之后文件夹下会多出一个 `dome.js`, 这时使用node进行运行就ok了

# ts-node

每次都要使用 tsc 进行转换，感觉很麻烦，所以有这样一个专门执行ts文件的好东西 ； ts-node 要比 node 执行的时间要慢些，因为是在转换止呕给出的结果

npm install -g ts-node

示范：
ts-node dome.ts
